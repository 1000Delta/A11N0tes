# shell字符串颜色

## 颜色语法

`\033[{颜色};{颜色}m{要输出的内容}\033[0m`

两个颜色分别可以指定前景色和背景色。可以仅使用前景色或后景色，即省略`;`

`\033[颜色m要输出的内容\033[0m`

|  34  | 蓝     |
