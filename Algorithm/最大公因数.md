# 最大公因数和最小公倍数

## 最大公因数

### 欧几里德算法

> 先求两数的余数，然后用较小的数除以余数，一直求到余数为0，此时较小的数就是两数的最大公约数。

C语言基本实现：

```c
int gcd (int a, int b) {

    int t;
    if(a < b) {

        t = a;
        a = b;
        b = t;
    }
    while(b > 0) {

        t = a % b;
        a = b;
        b = t;
    }
    return a;
}
```
