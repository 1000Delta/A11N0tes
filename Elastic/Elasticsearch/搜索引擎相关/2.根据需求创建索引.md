# 根据需求创建索引

## 明确需求

索引名：link

数据类型名：webPage

字段：

- keyword 关键字
- title 标题
- url 地址
- picurl 图片地址
- intro 简介
- content 内容
- tags 标签
- note 备注

## 创建新索引

创建索引时需要指定主，副本分片数量

选择3个主分片，一个副本分片

并且同时创建url分析器，便于对url字段进行分析

url分析正则：

`^(?:http|https|ftp)\:\/\/(\w+\.(\w+\.\w*))(\/[^#\?\s]*)`

`^(?:http|https|ftp)\:\/\/(\w+)\.(\w+\.\w*)\/[^#\?\s]*`

在索引级别使用soerce强制数据类型转换

```json
PUT /link
{
    "settings": {
        "index": {
            "number_of_shards": 3,
            "number_of_replicas": 1
        },
        "analysis": {
            "analyzer": {
                "url_analyzer": {
                  "type": "custom",
                  "tokenizer": "uax_url_email"
                }
            }
        },
        "index.mapping.coerce": false
    }
}
```

## 设置分析器

对于特殊数据类型，需要使用自定义分析器来对文本进行分词

url类型，使用自定义分析器如下：

```json
PUT /link/_settings
{
  "analysis": {
    "filter": {
      "url": {
        "type": "pattern_capture",
        "preserve_original": true,
        "patterns": [
          "^(http|https|ftp)\\:\\/\\/",
          "(?:(\\w+)\\.)?\\w+\\.\\w*\\/",
          "(?:\\w+\\.)?(\\w+\\.\\w*)\\/[^#\\?\\s]*"
          ]
      }
    },
    "analyzer": {
      "url_analyzer": {
        "type": "custom",
        "tokenizer": "uax_url_email",
        "filter": ["url"]
      }
    }
  }
}
```

将URL拆分成协议，子域名，域名三部分。

## 添加数据类型

原型：

```json
PUT /link/_mapping/webPage 
{
    "properties": {
        "keyword": {
            "type": "keyword",
             //"index": "not_analyzed" // 关键字不拆分分析
        },
        "title": {"type": "text"},
        "url": {
            "type": "text",
            "analyzer": "url_analyzer" // 采用自定义url分析器
        },
        "picurl":{
            "type": "text",
        	"index": "no" // 图片url不可搜索，不索引
        },
        "intro": {"type": "text"},
        "content": {"type": "text"},
        "tags": {
            "type": "keyword",
            //"index": "not_analyzed" // 标签不拆分分析
        }, 
        "note": {"type": "text"}
    }
}
```

正确语法：

```json
PUT /link/_mapping/webPage 
{
    "properties": {
        "keyword": {
            "type": "keyword"
        },
        "title": {"type": "text"},
        "url": {
            "type": "text",
            "analyzer": "url_analyzer" 
        },
        "picurl":{
            "type": "text",
        	"index": "no"
        },
        "intro": {"type": "text"},
        "content": {"type": "text"},
        "tags": {
            "type": "keyword"
        }, 
        "note": {"type": "text"}
    }
}
```



## 创建索引的同时创建数据类型

```json
PUT /link
{
    "settings": {
        "index": {
            "number_of_shards": 3,
            "number_of_replicas": 1
        },
        "analysis": {
            "analyzer": {
                "url_analyzer": {
                  "type": "custom",
                  "tokenizer": "uax_url_email"
                }
            }
        },
        "index.mapping.coerce": false
    },
    "mappings": {
        "webPage": {
            "properties": {
                "keyword": {
                    "type": "keyword",
                     //"index": "not_analyzed" // 关键字不拆分分析
                },
                "title": {"type": "text"},
                "url": {
                    "type": "text",
                    "analyzer": "uax_email_urls" // 采用url分析方式
                },
                "picurl":{
                    "type": "text",
                    "index": "no" // 图片url不可搜索，不索引
                },
                "intro": {"type": "text"},
                "content": {"type": "text"},
                "tags": {
                    "type": "keyword",
                    //"index": "not_analyzed" // 标签不拆分分析
                }, 
                "note": {"type": "text"}
            }
        }
    }
}
```
