## 渲染元素内容

```vue
<div id="id">
    {{ message }}
</div>
```

`{{ message }}`将会被替换成要渲染的内容



## 绑定元素特性

绑定元素特性要用到`v_bind`指令

例：

```vue
<div id="id">
    <span v-bind:title="message">悬停鼠标显示信息</span>
</div>
```

绑定了一条`title`特性

`v-bind:title` 将这个元素节点的`title`特性和Vue实例的`message`属性保持一致



## 渲染项目列表

```vue
<div id="list">
    <ol>
        <li v-for="todo in todos">
            {{ todo.text }}
        </li>
	</ol>
</div>
```

```javascript
var list = new Vue ({
    el: '#app',
    data: {
        todos: [
            {text: 'first'},
            {text: 'second'},
            {text: 'third'}
        ]
    }
})
```

通过`list.todos.push({text: content})`的方法可以添加新内容
