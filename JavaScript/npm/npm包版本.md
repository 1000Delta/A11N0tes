# npm包版本

```json
{
    "devDependencies": {
        "axios": "^0.18",
        "bootstrap": "^4.0.0",
        "cross-env": "^5.1",
        "jquery": "^3.2",
        "laravel-mix": "^2.0",
        "lodash": "^4.17.5",
        "popper.js": "^1.12",
        "vue": "^2.5.7",
        "element-ui": "^1.2.8",
        "vue-loader": "^11.3.4",
        "vue-router": "^2.4.0"
    }
}
```

- `~` 会匹配最近的小版本依赖包
  - `~1.2.3` 匹配 `1.2.x` 但是不匹配`1.3.0`
- `^`匹配最新的大版本依赖包
  - `^1.0.0`会匹配所有`1.x.x`, 但不会匹配`2.0.0`
- `*` 安装最新版本的安装包
