# go 命令

## go get

拉取、安装远程包

```sh
go get SOURCE
```

需要本地安装有所需的版本管理工具（svn、git等）

| 附加参数  | 备  注                                 |
| --------- | -------------------------------------- |
| -v        | 显示操作流程的日志及信息，方便检查错误 |
| -u        | 下载丢失的包，但不会更新已经存在的包   |
| -d        | 只下载，不安装                         |
| -insecure | 允许使用不安全的 HTTP 方式进行下载操作 |
